Introduction

Lotide is a mini Lodash clone that I created as a part of the Lighthouse Labs Web Development Bootcamp. It is a Javascript library that provides utilities to simplify operations with arrays, numbers, objects, strings, etc.

Get Started

Follow the following steps to get started on Lotide:

  1. To install Lotide run the following command:
    npm install @chenpoyi/lotide
  2. To require Lotide run the following command:
    const _ = require('@chenpoyi/lotide');
  3. To call Lotide functions run the following command:

    const results = _.tail([1, 2, 3]) // => [2, 3]

assertArraysEqual

assertArraysEqual(): determines if two arrays are equal and outputs appropriate messages.

Usage
const first = [1,2,3];
const second = [1,2,3];
const third = [1,2,'3'];
assertArraysEqual(first, second); //Should return true
assertArraysEqual(first, third); //Should return false
assertEqual

assertEqual(): determines if two primitives are equal and outputs appropriate messages.

Usage

assertEqual("Lighthouse Labs", "Bootcamp"); //Should return false
assertEqual("Hi","Hi"); //Should return true
assertEqual(1, 1); //Should return true
assertEqual(1,2); //Should return false

assertObjectsEqual

assertObjectsEqual(): determines if two objects are equal and outputs appropriate messages.

Usage

const obj1 = {a: '123', b: '567'};
const obj2 = {a: '123', b: '567'};
const obj3 = {a: '123', c: '567'};
assertObjectsEqual(obj1, obj2); //Should return true
assertObjectsEqual(obj1, obj3); //Should return false

countOnly

countOnly(): returns the number of occurences of a specific item in an array.

Usage

const firstNames = [ "Karl", "Salima", "Agouhanna", "Fang", "Kavith", "Jason", "Salima", "Fang", "Joe" ];
countOnly(firstNames, { "Jason": true, "Karima": true, "Fang": true, "Agouhanna": false }); //Should equal undefined.
countOnly(firstNames, { "Jason": true, "Karima": true, "Fang": true, "Agouhanna": false }); //Should equal to 2.
